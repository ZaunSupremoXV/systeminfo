# Systeminfo
 
S.O: Windows

Commands:

    npm install

    npm start



Commands builder:

    npm build-installer or npx electron-builder -g



System Resources:

    - Model

    - Manufacturer

    - Serial

    - Hostname
    
    - User

    - Network Status
    
    - CPU processing (%)

    - Memory usage (%)

    - Total memory (GB)
    
    - Foreground process

    - Foreground program
